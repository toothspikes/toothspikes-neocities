Since this is my first post, itll probably be a lot more informal than the next posts, but I don't think any of this website can pass for formal anyways.   • I've juggled the idea of going back to college, or I guess just continuing college and getting my bachelors, but I think my best move is to branch out on my own now. Hopefully now I can stop using school as an excuse to not study topics outside of it.
  • If I do a wordpress site, I think I'll keep it simple and give it a modern sleek look, basically making it a resume/portfolio page. I'll also likely use it on actual resumes when applying for jobs or for clients to see when freelancing. If I get confident enough on wordpress, I can start creating themes from scratch and uploading them to be used by others.

I know this isn't anything monumental to start out with, but my focus has been on actually getting the blog page looking good and refreshed. I'm sure it'll end up changing again or I'll be adding tweaks here and there but for now, I think I like how it turned out. Stay turned for more! --K

I will be also testing my content on different browsers, as well as on my phone and having others help when it comes to tablets.
